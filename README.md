# Soil Monitoring System

A Python Tkinter project to monitor soil health and assist in smart farming.

## Features

- Simulates real-time soil sensor data:  
  - Nitrogen (N)  
  - Phosphorus (P)  
  - Potassium (K)  
  - Temperature (Â°C)  
  - pH  
  - Humidity (%)  
- Displays graphs for trends and history of soil parameters  
- Provides alerts if soil conditions are outside safe ranges  
- Simple GUI built with Tkinter and ttkbootstrap  
- Crop and disease prediction placeholders for future updates
